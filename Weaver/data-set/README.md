# Dataset Guide

## Regression Datasets

1. california_housing.csv (20,640 rows)

- Target: MedHouseVal (median house value)
- Features: 8 numeric (population, income, location, etc.)
- Test with: Linear Regression, Decision Tree, Random Forest

## Classification Datasets

2. iris.csv (150 rows) - 3 classes

- Target: target (setosa, versicolor, virginica)
- Features: 4 numeric (sepal/petal dimensions)
- Test with: Logistic Regression, Decision Tree, Random Forest

3. breast_cancer.csv (569 rows) - Binary

- Target: target (0=benign, 1=malignant)
- Features: 30 numeric
- Test with: Logistic Regression, Decision Tree, Random Forest

4. wine.csv (178 rows) - 3 classes

- Target: target (wine class 0, 1, 2)
- Features: 13 numeric (alcohol, color intensity, etc.)
- Test with: Logistic Regression, Decision Tree, Random Forest

5. train.csv (Titanic) - Binary

- Target: Survived (0, 1)
- Features: Pclass, Age, SibSp, Parch, Fare
- Test with: Logistic Regression, Decision Tree, Random Forest

All datasets are clean, CSV format, and ready to upload through the UI!
